Nurture Nexus

Nurture Nexus is a comprehensive platform designed to help parents manage their children's development, education, and health, while also supporting parental well-being. The platform provides resources, diet plans, fitness exercises, and postnatal care tips in a single user-friendly interface.

Features

User Authentication: Sign-up, sign-in, and password reset.

Resource Library: Access parenting and educational resources.

Child Diet Plans: View and manage diet plans for children.

Postnatal Care Tips: Guidance and tips for new mothers.

Fitness Exercises: Personalized exercises for parents.

Technology Stack

Frontend: HTML, CSS, JavaScript

Backend: Python (Django)

Database: MySQL

IDE: VS Code
